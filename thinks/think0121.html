# 1.21:头文件、源文件

为什么C语言要分. h和. c文件，这样有什么好处？

  1. 能不能把函数及变量的声明放在源文件里？
  2. 能不能把函数和变量的定义放在头文件里？
  3. 能不能在两个头文件里都声明同一个函数或者变量？
  4. 能不能在两个源文件里都定义同一个函数或者变量？
